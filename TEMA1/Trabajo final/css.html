¿Qué es CSS y cómo se programa?

CSS (Cascading Style Sheets) es un lenguaje utilizado para definir el estilo y diseño visual de las páginas web. Permite aplicar colores, fuentes, márgenes, tamaños y posiciones a los elementos HTML, transformando estructuras básicas en diseños atractivos y responsivos.

¿Cómo se programa en CSS?

CSS puede aplicarse de tres formas: en línea, dentro de una hoja de estilo interna en el archivo HTML, o mediante hojas de estilo externas. Utiliza selectores como body o .clase para aplicar estilos.
